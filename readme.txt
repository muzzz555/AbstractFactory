Abstract Factory ( Object Creational )
- ซื้อกาแฟที่ร้านกาแฟ เราสั่งไปแล้วเราไม่ต้องสนใจว่า ร้านเขาจะทำอะไร
- Object ในแต่ละ Object ขั้นตอนการสร้างมันเยอะและซ้ำขั้นตอนเดิมๆ แทนที่จะสร้าง Object เอง เราส่งให้คนอื่นทำ เช่น ร้านค้า หรือโรงงาน

ข้อดี
- class ที่ผลิตด้านในมีความยุ่งยากอะไรเราไม่สน เราเรียกใช้งานก็พอ
- มีความเเน่นอนของ Object ไม่ว่าจะสร้างกี่ครั้งก็ได้ออกมาเหมือนเดิม ปรับยังไงก็ไม่พัง
- เปิด product ใหม่ได้ไม่ยาก
ข้อเสีย
- เปิด product ย่อยยาก มีปัญหา
